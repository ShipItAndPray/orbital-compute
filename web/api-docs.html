API Documentation

REST API for the Orbital Compute Simulator. Base URL: https://api.orbital-compute.dev/v1

Setup

# Install and start the API server pip install orbital-compute-sim orbital-compute serve --port 8080 # Or use Docker docker run -p 8080:8080 orbital-compute/sim:latest # Authentication # Include your API key in the Authorization header: curl -H "Authorization: Bearer YOUR_API_KEY" \ https://api.orbital-compute.dev/v1/fleet/status
POST /simulate Run a constellation simulation

Request Body

{ "num_satellites": 12, "sim_hours": 24, "num_jobs": 100, "altitude_km": 550, "inclination_deg": 53, "solar_watts": 2000, "battery_wh": 5000 }

curl Example

curl -X POST https://api.orbital-compute.dev/v1/simulate \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{"num_satellites": 12, "sim_hours": 24, "num_jobs": 100}'

Response 200

{ "simulation_id": "sim_a1b2c3d4", "status": "completed", "stats": { "fleet_utilization": 68.4, "completed_jobs": 97, "avg_latency_sec": 142.3, "avg_battery_pct": 72.1, "eclipse_fraction": 35.2, "orbital_period_min": 95.6 }, "satellite_count": 12, "timesteps": 906 }
GET /fleet/status Get current fleet status

curl Example

curl https://api.orbital-compute.dev/v1/fleet/status \ -H "Authorization: Bearer YOUR_API_KEY"

Response 200

{ "fleet_id": "fleet_001", "satellites": [ { "id": "SAT-001", "status": "computing", "position": { "lat": 32.4, "lon": -118.2, "alt_km": 550 }, "battery_pct": 78.3, "temp_c": 42, "in_eclipse": false, "active_jobs": 3, "ground_contact": "Hawaii" } ], "total_active_jobs": 24, "fleet_utilization": 72.1 }
GET /satellites/{id} Get detailed satellite telemetry

Parameters

idSatellite identifier (e.g., SAT-001)
?fromStart time ISO 8601 (optional)
?toEnd time ISO 8601 (optional)

curl Example

curl https://api.orbital-compute.dev/v1/satellites/SAT-001 \ -H "Authorization: Bearer YOUR_API_KEY"

Response 200

{ "id": "SAT-001", "orbital_elements": { "altitude_km": 550, "inclination_deg": 53, "raan_deg": 0 }, "hardware": { "gpu_count": 8, "gpu_type": "A100-space", "solar_watts": 2000, "battery_wh": 5000 }, "telemetry": { "battery_pct": 78.3, "temp_c": 42, "solar_input_w": 1850, "compute_load_w": 800 }, "uptime_hours": 4320, "total_jobs_completed": 12847 }
POST /jobs Submit a compute job

Request Body

{ "name": "training-run-042", "gpu_count": 4, "estimated_duration_sec": 3600, "priority": "high", "container_image": "registry.orbital-compute.dev/pytorch:2.1", "data_input_gb": 2.5 }

curl Example

curl -X POST https://api.orbital-compute.dev/v1/jobs \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_API_KEY" \ -d '{"name":"training-run-042","gpu_count":4,"estimated_duration_sec":3600,"priority":"high"}'

Response 201

{ "job_id": "job_x7y8z9", "status": "queued", "assigned_satellite": "SAT-003", "estimated_start": "2025-01-15T14:22:00Z", "queue_position": 2 }

curl Example

Response 200

{ "job_id": "job_x7y8z9", "name": "training-run-042", "status": "completed", "assigned_satellite": "SAT-003", "started_at": "2025-01-15T14:22:00Z", "completed_at": "2025-01-15T15:18:42Z", "gpu_hours_used": 3.78, "cost_usd": 11.34, "output_url": "s3://orbital-results/job_x7y8z9/output.tar.gz" }

Error Codes

CodeMeaningDescription
400Bad RequestInvalid parameters or missing required fields
401UnauthorizedMissing or invalid API key
403ForbiddenInsufficient permissions for this resource
404Not FoundResource does not exist
429Rate LimitedToo many requests. Retry after indicated delay
500Internal ErrorServer error. Contact support if persistent

Rate Limits

All endpoints are rate-limited per API key:

  • GET endpoints: 100 requests/minute
  • POST /simulate: 10 requests/minute
  • POST /jobs: 30 requests/minute
